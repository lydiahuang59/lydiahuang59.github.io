I first started coding in summer of 2018. Here are some of the things I have worked on since then!

Design Hierarchy Explorer for Verilog
File explorer for Verilog files. Reads a directory of Verilog files and figures out its design hierarchy. Done as a part of my Xi'An Technology summer internship in 2018.
Github link: https://github.com/lydiahuang59/XiST-design-explorer.git

Daily Diary/Mood Tracker
Simple text diary Android app, where users can write daily entries and also specify their mood for the day. Users can choose from multiple emotions and choose the strength of those emotions. I'm currently trying to implement an learning algorithm that can figure out a user's mood based on how they typed their current entry.
Github link: https://github.com/lydiahuang59/Mood-Diary-Android.git
